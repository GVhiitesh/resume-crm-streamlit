# Resume Management CRM

This is a role-based Resume Management System built using **Streamlit** and **SQLite**.

## Features
- Admin & Staff login
- Role-based access control
- Add, view, edit, delete resumes
- Filter and search resumes
- Export filtered data to Excel
- Dashboard metrics
- Secure password hashing

## Tech Stack
- Python
- Streamlit
- SQLite
- Pandas
